# Loan Performance & Securitisation Risk Analysis Dashboard

## Overview

This project is an advanced Power BI dashboard designed to analyze loan portfolio performance, credit risk, securitisation exposure, expected credit loss (ECL), and stress-testing scenarios. The dashboard provides actionable insights for investors, financial analysts, and risk management teams by identifying high-risk borrowers, default patterns, portfolio concentration risks, and potential financial losses under different market conditions.

## Key Features

* Loan Performance & Default Analysis
* Expected Credit Loss (ECL) Calculation
* Risk Categorization (Low, Medium, High Risk)
* Securitisation Portfolio Assessment
* Regional Risk Analysis
* Investor Risk Monitoring Dashboard
* Stress Testing & Scenario Simulation
* Interactive Filters and Dynamic Visualizations

## Insights Generated

* Small business loans exhibit the highest default risk.
* High-risk borrowers show significantly higher default probabilities.
* Higher debt-to-income ratios correlate with increased defaults.
* Medium and high-risk segments contribute most to portfolio losses.
* Portfolio performance can be evaluated under Normal, High Interest, and Crisis scenarios.

## Tools & Technologies

* Power BI
* DAX
* Data Modeling
* Data Visualization
* Financial Risk Analytics
* Credit Risk Assessment
* Securitisation Analysis

## Business Impact

This dashboard enables financial institutions and investors to assess portfolio quality, monitor credit risk exposure, estimate expected losses, and make data-driven lending and investment decisions through advanced risk analytics and stress-testing simulations.
